Interactive Training Session

Stores Inventory Management System

A detailed, button-by-button presentation detailing the layout, modules, and processes of the NSIMs platform for Store Officers.

Module 1.0

Global UI Layout & Controls

Familiarize yourself with the common layouts, widgets, and header bars that are accessible from any system pane.

  • Navigation Sidebar: Resides on the left side. Shows your name, avatar, and allows switching between modules.
  • Theme Toggle: Swaps the interface instantly between Light and Dark mode options.
  • Notification Center: Lists low stock status alerts and expirations in real time.
  • System Zoom: Standardizes scaling of all widgets, grids, and charts across screens.
Interface Button / Control Actions Performed
Sidebar Toggle Collapses the left navigation sidebar into a compact view to maximize data space.
Zoom (+ / -) Scales the interface elements up or down (50% to 200%) to accommodate different monitors.
Notification Center Toggles the notification alerts pane showing stock warnings.
localhost:8000/dashboard
Module 1.1

Store Officer Dashboard

Your operational center. Evaluates key logistics numbers and visualizes daily inventory stats.

  • Analytical Insights: Shows inventory metrics (Total Balance, Daily Issuance, Low Stock alert indicators).
  • ApexCharts Panels: Weekly/Monthly activity series graphs mapping stock items against variances.
  • Recent Transactions Feed: Lists the latest 4 operations showing entry date, supplier details, and statuses.
Interactive Control Actions Performed
Refresh Dashboard Clears the cached metrics values and updates stats directly from database records.
Chat Button Opens secure communication console to speak with administrators or other colleagues.
Moon Icon Toggles dark mode theme with glassmorphism shading.
localhost:8000/dashboard
Module 1.2

Received Items Log

The digital stock ledger that monitors incoming inventory batches and supplier deliveries.

  • Grid columns: Tracks Description, Received Qty, Stock Balance, Variance, and Statuses (Full/Partial/Donor).
  • Variance Highlighters: Color-coded numbers (green for matches, orange for partial, red for discrepancies).
  • Quick Filter Filters: Filter with one click by Partials, Pending, or Baseline Stock.
Interactive Control Actions Performed
View Details Opens preview details modal listing batch details, serial numbers, and driver phone info.
Refresh List Reloads the incoming stock table to show the latest records.
Filter / Reset Applies current dropdown filters or resets forms to search queries.
Continue Delivery Only visible on partial deliveries. Reopens the intake form to enter the remaining balance of items.
localhost:8000/received-items
Module 1.3

Add Inventory Item (Intake)

The primary form for adding new batches into inventory.

  • Batch Header Section: Select ledger category and search/select suppliers.
  • Driver Details: Record contact person details, driver names, phone numbers, and vehicle numbers.
  • Mark as Donor: A toggle switch that adjusts inputs for donations.
  • Dynamic Item List: Record descriptions, received quantities, and physical counts.
  • S/N Generation: Dynamic input generation based on quantity, with support for bulk paste.
Form Control Button Actions Performed
Bulk Paste S/N Opens textarea popup to paste multiple comma-separated serial numbers, auto-populating fields.
Mark as Donor Toggles interface fields to collect donor name and vouchers instead of purchase orders.
Submit for Approval Validates fields, saves stock entry draft, and submits it to Head of Stores.
localhost:8000/inventory/create
Intake / Stock Entry Form Layout Contains Ledge Category, Supplier Select2, Delivery Driver Inputs, and dynamic rows for adding multiple inventory items.
Module 1.4

Issued Items Log

Disbursement tracking system recording all items leaving the store.

  • Detailed Logging: Shows Timeline, Item Category, Destination Department, Collector Name, and Store Officer.
  • Loan Recovery Indicators: Highlights temporary loans in orange; lists them as active until returned.
  • Stats Insight cards: Shows Total Issued, Permanent allocations, and Active Loans.
Grid Filter Control Actions Performed
Live Search Input Filters table data instantly as you type by item name, department, or collector.
Category Filter Filters the log rows to show items from a specific category.
Issue Type selector Toggles listing rows between Permanent Allocations and Temporary Loans.
localhost:8000/issue-items
Module 1.5

Asset Recovery & Returns

Process recovered assets and return them to store stock.

  • Outstanding Holdings Grid: Lists all items currently out on temporary loan.
  • Overdue Alerters: Shows flashing red warning badges for items past their expected return date.
  • Return History Bottom Sheet: Displays the complete history of recovered assets.
Recovery Action Button Actions Performed
Recovery Button Opens the recovery panel modal to record returned items.
Max Qty Shortcut Sets the return input directly to the loan's maximum outstanding balance.
Return History Toggles the Return History log bottom sheet panel.
Finalize Return Submits the recovery record, updates inventory stock, and logs the item status.
localhost:8000/returns
Module 1.6

Requisitions & Collections

Manage internal requisitions and confirm physical item collection.

  • Pending Approvals Tracker: Lists requests currently routing through HOD/Authorizer steps.
  • Collection Action Control: Once approved, the Store Officer uses this view to confirm item disbursement.
  • Requisition Summary: Displays requested vs approved quantities, serial numbers, and remarks.
Requisition Button Action Actions Performed
View Detail Opens a detailed dialog showing item requests, approvals, and timeline details.
Confirm Collection Confirms physical receipt of items. Updates stock balance and logs the transaction.
New Requisition Opens the checkout panel to request items from stores.
localhost:8000/personnel/requisitions
Module 1.7

Stock Check & Verification

Perform physical stock counts, log discrepancies, and update the ledger.

  • Audit Grid: Shows Description, Received Qty, System Stock Balance, and Current Variance.
  • Bulk Verification: Check multiple items to verify them simultaneously.
  • Variance Calculator: Updates variance dynamically as you input physical counts.
  • Verification Report PDF: Generate professional audit reports directly from the interface.
Audit Verification Action Actions Performed
Verify Button Opens the stock verification modal to enter physical counts and remarks.
Print Audit List Generates a printable sheet for manual physical stock audits.
Seal Verification Record Saves physical count details and updates system stock balances.
Print Draft PDF Generates and downloads a formal Stock Verification Report PDF.
localhost:8000/stock-check
Module 1.8

Service SRA Creation

Log servicing, vehicle repairs, and non-inventory contracted jobs.

  • Header Meta Fields: Auto-populates Department, Station, and Region data.
  • Vehicle servicing details: Fields for vehicle numbers, LPO reference numbers, and A&E records.
  • Delivery radio selectors: Choose between full delivery and partial deliveries.
  • Details Log Area: Log parts used and details of the service performed.
Form Action Button Actions Performed
My SRAs Navigates to the listing page to track SRA approval statuses.
Submit for Approval Submits the Service SRA to the Head of Admin for approval.
localhost:8000/service-sra/create
Module 1.9

Communication Hub (Messages)

Coordinate approvals and discuss inventory details via secure internal messaging.

  • Chat Roster: Shows administrators, command center contacts, and colleagues.
  • Status indicators: Shows real-time online/offline statuses of personnel.
  • File uploads: Attach images, document files, or spreadsheets directly.
  • Security clearance indicators: Integrates timed rollback approval link notices.
Comms Action Control Actions Performed
Attachment Button Opens file select dialog to upload vouchers or spreadsheets.
Send button Transmits secure message text and attachments.
Encrypted Badge Displays security validation status.
localhost:8000/messages
Module 2.0

Settings & Digital Signatures

Manage profile settings, security credentials, and your official digital signature.

  • Signature uploads: Upload PNG signatures to sign off on vouchers and receipts automatically.
  • Credentials Management: Change your current login password.
  • Display Preferences: Toggle interface zoom scaling and glassmorphism styling.
Settings Navigation / Control Actions Performed
Change Photo Opens local folder to upload a new profile photo.
Upload Signature Uploads a signature image file (PNG/JPG) for document processing.
Remove Signature Deletes the stored signature from the database.
Update My Profile Saves name, phone numbers, and department changes to database.
localhost:8000/settings
